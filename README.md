# "RMIT" Car Repair Shop

A management system for a car repair shop made using Python.

#### Dependencies

This program is built in Python v3.13.12 environment, and does not require additional Python libraries.

#### Software Requirements

_The program should:_

- Allow shop's mechanics and receptionists to manage jobs for new or existing customers.
- Store all customers' data including name, membership, and service count.
- Print customer receipts for specific service jobs.
- Have presets for services (along with their hours) and parts (along with their prices).
- Be able to handle invalid inputs.
- Be operable using a menu.
- Allow users to modify existing presets of serices, and add, remove, or update parts information.
- Be able to display existing customers data, services presets, and parts presets.

---

#### Features

[] **Job Creation/Perform a Service** – Mechanics and receptionists can create a new service job by entering a customer name and their requested service with specific hours and parts when necessary.
[] **Customer Receipts** – Calculate the total cost of a job for the customer using prices from the presets for services and parts. Apply discounts for member customers. Print a receipt for the customer for the specific service job.
[] **Input Validation** – Handle user input validation at all stages.
[] **Program Menu** – Print a formatted menu and provide options to operate the software.
[] **Update Services and Parts Presets** – Modify the default value of services' hour preset (either a new default value or allow user input for specific hours). Add new parts to the preset, and remove and update the existing parts data.
[] **Display Stored Data** – Print existing customers list (including their membership status), and services and parts presets.
